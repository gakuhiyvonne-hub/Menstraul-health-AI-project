## Project overview 
This project uses an AI tool that predicts menstrual pain level based on ones sysmptoms. 
The user enters their details and the AI gives a predicted pain score between 1 and 10.

## DATASET
17,976 menstrual cycles from kaggle
source: https://www.kaggle.com/datasets/puspitachowdhury2/menstrual-health-dataset

## WHAT THE AI LOOKS AT 
- PMS symptoms (yes/no)
-  Flow level (light/moderate/heavy)
-  Mood score (1 - 10)
-  stress level(1 -10)

## RESULTS
- Average pain level: 4.85/10
- Heavy flow pain: 7.32/10 vs light flow: 3.32/10
